# Community Plugins

**Canonical repo for install / review / releases:**

https://github.com/diagonalcounty/blinks

Obsidian Community Plugins require `manifest.json`, `main.js`, and `README.md` at the **repository root**, with GitHub Releases tagged exactly as `manifest.json` → `version` (e.g. `0.2.0`).

This monorepo folder is a workshop snapshot. Prefer the dedicated repo for PRs and issues.
